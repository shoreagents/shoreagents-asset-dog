## Notes:
- Item codes will be auto-generated in the format: INV-XXX-SA
- You can leave optional fields (description, location, supplier, brand, model, SKU, barcode, remarks) empty if needed
- Initial Stock can be set to 0 if you want to add stock later via transactions
- Min/Max Stock Levels are optional but recommended for inventory management
- Unit Cost is optional but useful for cost tracking

## Testing Transfer Functionality:
1. Create at least 2 items (e.g., Item 1 and Item 2)
2. Add some initial stock to both
3. Use TRANSFER transaction type to move stock from Item 1 to Item 2
4. Check transaction history to see the linked transactions
